##city selection是国内外城市选择前端界面

相比传统的城市选择界面它的特点是：
	优秀的使用体验，城市选择一目了然，即时展现
	细分城市级别，获取城市精准匹配
	支持修改输入数据格式化方法，支持输出需求格式化数据？！
兼容的浏览器：IE10+、Chrome、Firefox、Safari，opera

## 安装

```shell
git clone https://github.com/zlstone/city_selection.git
```
